Introducting ChessBot

Apr 26, 2018

ChessBot is an elegant chess app with powerful features.

  • Basic move operations and outcome detection
  • A strong chess engine which offers hint for the next move
  • Human vs Human/Computer modes
  • PGN files import and export
  • Game saving and sharing
  • Encyclopedia of Chess Openings
  • Play with friends via iMessages
  • Preview pgn files in iOS File app

And more features will be added in the future.
Get it from App Store and try it now.

If you have any suggestions or complaints, please feel free to tell us.
